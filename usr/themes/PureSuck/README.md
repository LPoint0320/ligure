# Ligure Theme for Typecho

> 干净，纯洁，淡雅朴素。
> Clean, pure & elegant.

基于 [PureSuck](https://github.com/MoXiaoXi233/PureSuck-theme) 1.3.2 定制，专为 [ligure.cc](https://ligure.cc) 打造。

## 特性

- 响应式三栏布局
- 深色/浅色主题切换（支持 ViewTransition 动画）
- Swup 无刷新页面过渡（PJAX）
- TOC 文章目录（侧边栏跟随）
- OwO 表情面板（B站 / MoXiify 表情）
- GitHub 卡片、Bilibili 视频嵌入
- 文章加密、字数统计、CC 协议版权
- 短代码：提示框、折叠面板、Tab、时间线、友链卡片
- 图片点击缩放（medium-zoom）
- 8 种配色方案

## 安装

1. 将本主题文件夹上传至 Typecho 的 usr/themes/ 目录
2. 在 Typecho 后台「设置」->「外观」中启用 Ligure 主题
3. 进入「设置外观」进行详细配置

## 依赖

- Typecho 1.2+
- PHP 7.4+
